# Sistema de Assistência Técnica Inteligente (IA)

Este projeto foca na otimização do processo de assistência técnica interna para motobombas. O sistema recebe evidências (fotos e descrições), auxilia o analista na triagem via IA e gera automaticamente relatórios de devolutiva.

## 🛠️ Tecnologias de Estudo
- **Frontend:** React
- **Backend:** FastAPI (Python)
- **IA:** Visão Computacional (Deep Learning)
- **Documentação:** LaTeX (Padrão SBC)

## 📋 Objetivos Principais
- Triagem automatizada de falhas motoras (70% dos casos).
- Padronização de laudos técnicos de garantia.
- Redução do tempo de resposta (Lead Time) do suporte interno.
